# 🌟 Doggles 🌟
---
📖 A chrome extension to simulate color blindness that dogs experience.
---

### 🚀 Features

* **Feature 1:** Deuteranopia
* **Feature 2:** Decreased Brightness Discrimination
* **Feature 3:** Adjustable Acuity

---
